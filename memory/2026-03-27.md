# 2026-03-27

- Anh Bảo chốt rule mới cho toàn bộ skill cào web visa: report mặc định gồm 6 phần business + 1 mục `TECHNICAL CHANGELOG`; bỏ `EXECUTIVE SUMMARY`; bỏ `TÓM TẮT THAY ĐỔI SO VỚI BASE / ĐẦU VÀO NỘI BỘ` trừ khi được yêu cầu riêng.
- Nếu không có thay đổi thực chất về giá visa, điều khoản, hồ sơ, hiệu lực, ngoại lệ quốc tịch... thì mục `TÓM TẮT THAY ĐỔI SO VỚI HÔM QUA` phải ghi đúng câu: `Không phát hiện sự thay đổi so với báo cáo hôm qua`.
- Các thay đổi kỹ thuật như `0 -> 00` phải đưa sang mục `TECHNICAL CHANGELOG`, không đưa vào phần tóm tắt thay đổi business.
